# Install RPT Fonts

The RPT font files are available in your GitHub repository at:
https://github.com/yasseralotaiibi/riyadh-metro-website/tree/main/fonts

## Available Fonts

From your GitHub repository, you have these font files:

### Regular Weight:
- `alfont_com_rpt-Regular.ttf` (52,896 bytes)
- `alfont_com_rpt-Regular.woff2` (22,116 bytes)

### Bold Weight:
- `alfont_com_rpt-Bold.ttf` (51,380 bytes)
- `alfont_com_rpt-Bold.woff2` (21,124 bytes)

### Arabic Specific:
- `alfont_com_AlFont_com_-rpt-Regular.ttf` (52,940 bytes)
- `alfont_com_AlFont_com_-rpt-Regular.woff2` (22,084 bytes)
- `alfont_com_AlFont_com_-rpt-Bold.ttf` (51,424 bytes)
- `alfont_com_AlFont_com_-rpt-Bold.woff2` (21,116 bytes)

## Download URLs

You can download these fonts directly from:

```
https://raw.githubusercontent.com/yasseralotaiibi/riyadh-metro-website/main/fonts/alfont_com_rpt-Regular.woff2
https://raw.githubusercontent.com/yasseralotaiibi/riyadh-metro-website/main/fonts/alfont_com_rpt-Bold.woff2
https://raw.githubusercontent.com/yasseralotaiibi/riyadh-metro-website/main/fonts/alfont_com_AlFont_com_-rpt-Regular.woff2
https://raw.githubusercontent.com/yasseralotaiibi/riyadh-metro-website/main/fonts/alfont_com_AlFont_com_-rpt-Bold.woff2
```

## Installation

The fonts are now configured in `/src/styles/fonts.css` to use direct GitHub CDN URLs, so they will load automatically from your repository.

No manual installation needed - the fonts will load directly from GitHub!
